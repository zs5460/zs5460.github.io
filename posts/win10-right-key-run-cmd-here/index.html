win10右键菜单添加在此处运行CMD

Apr 3, 2016 in 电脑技巧
win10
1 min read

win10取消了右键“在此处运行CMD”，改成了powershell，这个东西虽然功能多，但我用不上，而且启动速度比CMD慢多了。那怎么恢复CMD呢？

  1. 打开注册表

  2. 依次展开：HKEY_CLASSES_ROOT\Directory\background\shell

  3. 新建项“Run CMD Here”，在右边新建字符串值“Icon”，值为“cmd.exe”。

  4. 新建子项“command”，默认值为cmd.exe /s /k pushd "%V"
